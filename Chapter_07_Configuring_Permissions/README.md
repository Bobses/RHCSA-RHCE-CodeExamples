Advanced Permissions
---------------------

    SUID        4   u+s -  as the owner of the file being executed
    GUID        2   g+s -  same group rights of the file being executed
    Sticky bit  1   +t


Work with users/groups/passwords
--------------------------------

    useradd
    groupadd
    usermod


Permissions
-------------

    umask -S
    chmod
    chown

ACL
----

    setfacl
